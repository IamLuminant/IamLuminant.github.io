Luminant's Expansions Plugin

A comprehensive Unreal Engine 5 plugin providing essential gameplay systems including inventory, items, interactions, and abilities.

Unreal Engine 5 C++ Blueprint Multiplayer

Modular Plugin System

Core Features

Inventory System

Advanced inventory management with support for:

  • Grid-based
  • Dynamic inventory
  • Example Drag & drop widgets
  • Tetris style item placement
  • Sorting and filtering

Item System

Flexible item framework featuring:

  • Modular item class
  • Example item types
  • Example item blueprints
  • Item stacking
  • Item Abilties

Interaction System

Comprehensive interaction framework with:

  • Item interactions
  • Example highlight feedback
  • Example UI feedback
  • Line Tracing
  • Timed interactions

Ability System

Powerful ability framework including:

  • Modular ability structure
  • Multiplayer support
  • Ability management component
  • Async Tasks
  • C++ and Blueprint freedom

Planned Features

Crafting System

Recipe-based crafting with resource management and skill progression.

Procedural Generation System

Dynamic world generation with biomes, structures, and cave generation.

Storage System

Persistent world storage with containers, chests, and player housing integration.

Water & Buoyancy System

Realistic water physics with swimming mechanics and floating objects.

Technical Details

Architecture
  • Component-Based: Uses UE5's component system for maximum flexibility
  • Event-Driven: Loose coupling through delegates and events
  • Object-Driven: Replicated objects for object driven code
Compatibility
  • Multiplayer Friendly: Fully replicated and replication friendly
  • Blueprint Friendly: Uses UE5's component system for maximum flexibility
  • Modular Design: Each system can be used independently
